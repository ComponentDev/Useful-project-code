退货政策

  • 1 对于所有在喜地购买的商品，在不影响二次销售的情况下，我们为您提供自签收之日起7天无理由退货服务。
  • 2 质量问题退货，请先与卖家联系，双方确认后，买家将商品、相关配件及发票退回，运费将与商品退款一同返回您的原支付账号。
  • 3 非质量问题，因客户误购、或对商品不满意等原因，寄回退货运费由客户承担，平台仅返回原商品本身的金额。
  • 4 喜地所售均为全新品，为保护消费者利益，以下商品不适用于7天无理由退货：

    • ( 1 ) 个人定制类商品；

    ( 2 ) 在线下载或者您拆封的音像制品，计算机软件等数字化商品；

    ( 3 ) 服装鞋帽类商品已洗涤的； 试穿后的内衣袜子；

    ( 4 ) 钻石、黄金、手表、珠宝首饰及个人配饰类商品;

    ( 5 ) 食品、保健品、个人护理用品、贴身用品、化妆品、虚拟商品;

    ( 6 ) 交付的报纸期刊类商品；

    ( 7 ) 无法保证退回商品完好的；

    ( 8 ) 其他根据商品性质不适宜退货，经您在购买时确认不宜退货的商品；

    ( 9 ) 部分备注不提供退换货服务的商品

    ( 10 ) 超过退换货规定时限的商品。

  • 5 如需换货，请您先申请退货，对需要换购的商品进行重新选择，购买；我们将对退货商品进行严格的收货审核及后续处理，以便提供给每一位消费者更公平的产品及服务，因此暂时无法给您提供直接的换货服务，希望得到您的支持。

    • ( 1 ) 图片及信息仅供参考，不属质量问题。因拍摄灯光及不同显示器色差等问题可能造成商品图片与实物有色差，一切以实物为准；

    ( 2 ) 由于您技术水平、使用环境限制造成的误报以及实际情况与您描述不符，喜地将不予受理，按原地址返回的运费由您承担。

退货流程

  • 1 如果您有退货的需求，可在确认收货7日内对符合退货条件的商品办理相应的退货申请，退货申请请在“喜地-个人中心-我的订单-对应订单”中选择“申请退款”。
  • 2 质量问题的退货，请详细描述问题，并上传图片至审核系统。
  • 3 所退商品须未经使用，原包装完好, 相关附属配件齐全，不影响二次销售。
  • 4 含有赠品、礼品包装（包装盒），请您务必连同要退换的货品一并退回，若无退回，将按照赠品的售价在退款中扣除。
  • 5 请您在系统通过退货退款申请审核后7天内完成发货并填写运单号，若因买家问题导致退款延时或无法退款的，买家承担责任。
  • 6 为提升我们的服务质量，喜地诚挚地邀请您在退货时填写退货原因。
  • 7 配送到达平台退货仓库并签收后，喜地进行商品审核，审核通过后将统一办理退款操作。查看退款规则
