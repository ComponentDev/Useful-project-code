登录喜地

点击页顶登录进入登录界面。没有账号的海客请先注册。

挑选商品

喜地为您精心挑选了10个国家的优质品牌，您可以进入国家馆，挑选您喜欢的国家和特色商品。

通过喜地的商品分类直接查找您需要的商品类别。 直接搜索您需要的商品。

加入购物车

挑选商品后，在列表页或商品详情页面点击 按钮，将商品放入购物车中。

在购物车中，您可以直接查看到商品的名称、价格及其他相关信息。

系统默认每件商品的订购数量为1件，如果您想购买多件商品，可修改购买数量。

您可以将商品加入收藏夹，或是选择移除。

填写收货信息和发票类型

请详细准确填写收货信息（收货人姓名、联系方式、收货地址），否则将会影响您订单的处理或配送。 填写过的地址，下次购物为默认地址，如若更换，您可以继续添加其他收货地址。

喜地默认开具个人发票，您可以选择填写其他类型。

支付货款

成功提交订单后，需3小时内完成付款，喜地支持支付宝和银联第三方支付平台支付。

收货评价

收到商品后，您就可以进行评价了。

1如何取得发票

( 1 )本网站所售商品都是正品行货，并由喜地电商平台统一为所销商品开具发票， 可用作单位报销凭证；

( 2 )由于喜地平台实现不包邮政策，支付时向您收取的配送费实际收取方为快递公司，平台只有具商品发票，因此您收到的发票金额不含配送费金额；

( 3 )您每次在喜地购买的商品，不论是否来自同一供应商，在无特殊情况下将由同一包裹发货，对应一张发票，发票金额为订单总金额；

( 4 )特殊情况下，如发生商品缺货，您将可以选择先发送有货商品，此时发票将对应该次实际发货的商品；您也可以选择待缺货商品补货后统一发货，此时发票将对应订单内的所有商品；

( 5 )发票将和您所购商品一同邮寄，请您在签收商品的同时不要遗忘发票。

2发票设置

( 1 )为了向顾客提供更好的质保服务，我们将按商品明细开具发票，包括但不限于商品名称、计量单位、数量、单价、大小写金额等项目，例：您在喜地购买了白雪牌中性笔G-909 0.38号中性笔（整盒装）1盒，发票内容将为“商品名称-白雪牌中性笔G-909 0.38号中性笔（整盒装），单位-盒，数量-1，单价-12.00”等。

( 2 )默认状态下我们将为您开具普通-个人发票。

( 3 )如需开具普通-单位发票，请在"单位名称"一项填写正确的公司抬头，抬头不能为空。

( 4 )如需开具增值税专用发票，请拨打客服热线 400-0066-200 了解相关规则流程。

( 5 )零售的烟、酒、食品、服装、鞋帽（不含劳保）、化妆品等消费品不能取得增值税专用发票（国税发 [2006] 156号）。

3售后涉票事宜

( 1 ) 收到发票后如发现发票抬头、内容或金额等与您下单时所填写的项目不符，请先与平台客服联系（点击申请换发票），换发申请审核通过后，客服人员将安排快递人员上门取回原发票，收到原件后，我们会在7个工作日内为您重新开具发票并寄出，来往邮寄费将由我们承担。

( 2 )如发生商品退货，无论是订单整单退货还是订单中某单一商品的退货，请将发票随同商品包裹一并寄回，以免给您退货带来不便。
