账户类

如何注册喜地？

点击页顶的注册进入注册页面，填写个人信息，点击“同意协议并注册”，完成注册。
登录个人邮箱，找到标题为“欢迎您加入喜地”的邮件，打开后点击“验证邮箱”即可。

温馨提示

( 1 ) 请务必填写正确有效的注册邮箱地址，否则当您忘记注册密码时，将无法成功找回，只能重新注册新用户。

( 2 ) 注册成功后，您可以修改邮箱地址、昵称、密码等个人信息。

( 3 ) 系统会发送验证邮件到您的注册邮箱，若您长时间未收到邮件，请点击“重新发送”。

( 4 ) 若您使用未验证成功的邮箱登录，点击登录后，系统会重新发送验证邮件到您的邮箱。

如何提高我账户的安全性？

您可以登录喜地账户， 在“我的喜地-账户信息”里，通过经常性修改密码、绑定账户到邮箱和手机的方式增加账户安全系数，保证账户安全。

登陆时显示账户异常怎么办？

若您的账户登陆时存在异常，请及时联系客服。

密码设置技巧有哪些？

为了您的账户安全，在设置密码时，请参考以下建议：

( 1 )密码由6-20位字符组成（字母区分大小写）。

( 2 )请使用数字、字母或符号的组合，如：gklfd_157、89712am%&等。

( 3 )果设置如下安全性较低的密码，系统会提醒您修改密码以保证安全性：

a全部由数字组成；

b全部由字母组成；

c全部由同一字符组成；

d与用户名或邮箱相同。

( 4 )定期修改密码，并做好记录工作，以免忘记。

( 5 )不要将支付密码和登录密码设置为同一密码。

验证邮箱手机的好处有哪些？

验证手机、邮箱后，当您的账户余额发生变动时，我们将第一时间短信或邮件通知您，保障您的资金安全。

购物类

订单提交成功后还可以修改订单信息吗？

很抱歉，订单一旦提交后将无法修改。订单生成后3小时未付款，系统自动取消订单。已付款订单如有紧急问题（如写错地址、电话等）请联系客服。

订单拒收后，退款什么时候返还？

如果订单进行了拒收，需要订单商品入仓后才能办理相关退款，您可以在拒收商品后联系客服进行处理，给您带来的不便请谅解！

商品包装是否有保证？

我们所发送商品均由专人进行打包，商品在未签收前都由我们负责，如在收到商品时发现包装有破损或是其它方面问题，请直接致电客服热线400-0066-200，客服人员会帮您解决。

为什么我的订单总是无法提交成功？

可能存在以下几种情况：

( 1 )订单信息填写不完整；

( 2 )订单商品库存不足或者库存无货；

( 3 )订单商品不能配送到您所填写的收货地址；

( 4 )网络延时以上各种情况都会在页面中弹出提示信息，可以通过修改订单信息（提示信息）或者稍后再试，即可成功提交订单。

在加入购物车后，是不是该商品已经被我预定了？

并不是，如果您想订购该商品，请把该商品加入购物车后进行结算。提交订单成功后，货物将在库存中锁定，若已锁定的库存在3小时内未完成支付，库存将重新释放。

支付不成功的原因有哪些？

( 1 )您的银行卡尚未开通网上银行支付功能，建议您到当地营业厅开通网上银行；

( 2 )所用银行卡超出该银行支持地域范围，请您更换银行卡试试；

( 3 )银行卡已过期、作废、挂失或者余额不足等，建议您咨询您的开户行；

( 4 )输入的银行卡号、密码或证件号等与预置的不符，建议您重新输入正确的卡密码或证件号等，如果您忘记了密码，请您及时与所属银行联系办理密码重置；

( 5 )银行系统数据传输可能出现异常，建议您刷新页面或者稍后重启电脑；

( 6 )您的网上银行支付有限额，详情请咨询银行客服热线；

( 7 )使用支付宝详情请咨询支付宝客服热线。

什么情况下回中止交易？

( 1 )您的订单提交成功后，三个小时内没有付款，默认交易中止；

( 2 )如果订单状态在配货中，喜地不支持交易中止，有特殊情况请联系客服；

( 3 )发货中如快递原因、天气原因等无法送达的，请联系客服；

( 4 )收到货之后，喜地支持7天内退货退款。

配送类

订单已提交成功，什么时候可以发货？

订单提交成功后我们会尽快发货，详细进度您可以进入“我的订单”，实时查看订单进度。
查看发货政策

订单已提交成功，可以改送到其他地方吗？

订单一旦提交成功，将无法修改。请在提交订单前仔细检查核对。如有紧急问题（如写错地址、电话等）请联系客服。

请问如何联系到给我送货的快递员？

建议您按照订单跟踪中的运单号在相关物流公司网站查询。

签收后可以当场验收商品吗？

您签收后，可以当场验收商品，如商品本身有问题请您在“我的订单”中提交退换货申请，将有专业售后人员为您解决。

签收时发现商品包装破损，该如何处理？

商品签收时如包装有破损，请您直接拒收；商品签收后如商品本身有问题请您在“我的喜地”中提交退换货申请，将有专业售后人员为您解决。

客户订单收取了运费，若后期发生退货的情况，之前的运费会退给客户吗？

如果是商品因质量问题产生退换货，喜地将返还运费，如果非商品质量问题产生的退换货，运费将不予返还。

快递告知丢件，怎么处理？

建议客户提交退款申请，如您还需要该商品请重新下单购买。

退款/退货类

喜地所售商品是否都支持7天无理由退货？

喜地部分商品不支持7天无理由退货，详情请您查看售后服务。其他符合条件的均默认支持7天无理由退货服务。

购买的商品出现质量问题或发错货如何处理？

喜地承诺符合售后退货规则的商品，由售出之日（以实际收货日期为准）起7日内可以退货，客户可以登陆进入“我的订单”提交退货申请。

为什么退款/退货申请没有审核通过？

您可在退款/退货详情页查看原因。具体请登录喜地账户，在“我的订单”查看相应记录。

网上退货申请提交后发现出错如何修改？

出现此类问题可以拨打客服电话400-0066-200进行咨询，由客服人员给您处理方案。

退货运费是如何收取的？

商品因质量问题发生的退货，由喜地承担退货运费；如果由于您的原因产生退货，需要您自己承担退货运费。

什么时候可以收到退款？

我们会第一时间为您办理退款，具体到款时间根据您的开户行不同略有差异，大概为1-7个工作日，详请联系您开卡行。详情见退款规则。

发票类

如何获取发票？

喜地所售商品都是正品行货，每张订单均自带中文机打的“商品专用发票”，此发票可用作单位报销凭证，下单时可自助开取。发票金额不含配送费金额。

如何设置发票信息？

请您在订单结算时的页面“发票信息”选项中设置发票信息。

开发票的注意事项？

( 1 )发票金额不能高于订单商品金额。

( 2 )商品发票开具为明细。

怎么换发票？

您在收到发票后如果发现因喜地原因，票据抬头、内容或金额错误，请您致电客服电话400-0066-200，由客服人员为您办理换发票事宜，往返快递费用由喜地承担。

如何开具增值税发票？

如需开具增值税专用发票，请拨打客服电话400-0066-200，了解相关规则流程。

客服类

如果我有问题或建议如何向喜地反馈？

如果您对商品或服务有不满，可以提出投诉 (仅限注册用户)。如果您对喜地有任何意见或建议，欢迎提交意见反馈，或将您姓名、联系方式及相关信息写明，发送至。喜地会尽快给予答复或处理。

客服电话的工作时间是？

400-0066-200的工作时间是周一至周日8:00-18:00（仅收市话费）。服务范围包括订单自助查询、订单服务、售后服务、意见与建议。未开通400服务地区的客户请拨打021-00000000。

如何联系在线客服？
